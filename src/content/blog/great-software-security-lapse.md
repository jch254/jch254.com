No shit Sherlock... here comes the Great 2025-26 Software Security Lapse affecting anything and everything that is powered by software from the vibers.

## The problem

While **vibe coding** — writing apps via conversational AI prompts — is the shiny new toy in the land of 1s and 0s, some serious alarm bells are clanging in the security crowd. Generative AI tools like Claude Code, Cursor, and GitHub Spark make it feel like coding is as easy as chatting, but experts warn this can open the door to unchecked vulnerabilities and data exposure.

## The reality

Sure, it's liberating for creativity and undeniably hot as a skill... but deploying code you didn't fully inspect? That's like building a house of cards on shaky ground. The vibes are nice, but zero-trust principles, secure prompting, and vigilant oversight can't be left out. In the dance of tech, the music might be AI-fuelled, but the moves still need human discipline.

Ever heard of the [12 Factor App](https://12factor.net/)?
